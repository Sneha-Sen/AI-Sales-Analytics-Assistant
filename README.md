# GenAI-Powered Sales Analytics & Business Intelligence System

# Overview

This project demonstrates how Generative AI can enhance traditional data analytics workflows by combining Python, SQL and prompt engineering to generate automated business insights, dashboards, and executive-level summaries.

It simulates a real-world analytics pipeline used in modern data teams where AI acts as a decision support layer on top of structured business data.

# Key Idea

Instead of manually analyzing data every time, this project shows how:

Well-designed prompts can transform raw KPI data into business-ready insights in seconds.

# Project Architecture

Raw Sales Data
→ Python Data Cleaning & EDA
→ Feature Engineering
→ Clean Dataset Export
→ SQL-Based Analytical Layer
→ KPI Generation
→ GenAI Prompt Engineering Layer
→ Automated Insights & Business Narratives
→ Interactive Dashboard

# Prompt Engineering Framework

This project uses structured prompting techniques:

- Zero-shot prompting → Direct analysis
- One-shot prompting → Guided format learning
- Few-shot prompting → Pattern-based insights
- Chain-of-thought prompting → Step-by-step reasoning
- Persona prompting → Executive-level storytelling

# Key Business Insights
- Revenue is driven by a small base of loyal customers
- Strong dependency on West region and Technology category
- Product concentration risk in top-selling SKUs
- Growth potential exists in underperforming regions (South)
- Business is stable but not diversified
